# @marineyachtradar/mayara-gui

Web GUI for MaYaRa marine radar integration.

## Usage

This package is consumed by:
- `mayara` (standalone server) - embeds GUI in binary
- `@marineyachtradar/signalk-wasm` (SignalK plugin)

Install as dependency:
```bash
npm install @marineyachtradar/mayara-gui
```

Files are in the package root (no dist folder).

## Development

Serve locally with any static server:
```bash
npx serve .
```
